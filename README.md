# Chars — локальный мессенджер на Flask

Версия: ```0.0.2```

---

## О проекте

Этот проект — учебный мессенджер, созданный для изучения:
- Веб-разработки на Flask
- WebSocket (Flask-SocketIO)
- Работы с MySQL
- Аутентификации и управления сессиями
- Реального времени обмена сообщениями

Проект является **локальным** и **не предназначен для публичного использования**. Это песочница для экспериментов и обучения.

## Запуск проекта

### 1. Клонирование репозитория
```bash
git clone https://github.com/terrachell/messager.git
cd messager
```

### 2. Создание виртуального окружения

```bash
python3 -m venv env
source env/bin/activate  # Для Linux/macOS
# или
env\Scripts\activate     # Для Windows
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка базы данных

Убедитесь, что MySQL запущен

Создайте базу данных messager

Импортируйте схему из database/schema.sql (если есть)

### 5. Настройка конфигурации

Отредактируйте параметры подключения к БД в app.py:

```python
app.config['MYSQL_HOST'] = '127.0.0.1'
app.config['MYSQL_USER'] = 'root'
app.config['MYSQL_PASSWORD'] = 'your_password'
app.config['MYSQL_DB'] = 'messager'
```

### 6. Запуск проекта
```bash
python app.py
```

Сервер запуститься на ip 127.0.0.1 и порте 5001. Также если добавить в аргумент host
в эту строчку

```python
socket.run(host='0.0.0.0')
```

можно локально запустить его в вашей сети, он будет доступен по указанному в терминале ip.
