# Messager — локальный мессенджер на Flask

Версия: **0.0.2**

---

## 📌 О проекте

Этот проект — учебный мессенджер, созданный для изучения:
- Веб-разработки на Flask
- WebSocket (Flask-SocketIO)
- Работы с MySQL
- Аутентификации и управления сессиями
- Реального времени обмена сообщениями

Проект является **локальным** и **не предназначен для публичного использования**. Это песочница для экспериментов и обучения.
